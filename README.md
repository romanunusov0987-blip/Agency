# Agency

Телеграм-бот на `python-telegram-bot` с поддержкой раскладов таро «Да / Нет», страницы поддержки и личного кабинета.

## Структура проекта

- `bot.py` — основная логика бота: меню таро, сценарий «Да / Нет» с раскрытием карты, обработчики поддержки и личного кабинета.
- `requirements.txt` — зависимости проекта.
- `img/` — демонстрационные изображения (не используются ботом, но оставлены в репозитории).
- `README.md` — текущее описание проекта и структуры.

## Быстрый старт

1. Установите зависимости: `pip install -r requirements.txt`.
2. Задайте токен бота: `export TELEGRAM_BOT_TOKEN=<token>`. При необходимости укажите ссылки на поддержку и консультации через переменные `SUPPORT_CHAT_URL` и `CONSULTATION_URL`.
3. Запустите: `python bot.py`.

Для корректной работы сценария таро подготовьте изображения карт в `images/back.png` и `images/faces/<id>.png` (0–77).
